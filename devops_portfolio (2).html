About Me

DevOps Engineer & Cloud Migration Specialist

I'm a passionate DevOps Engineer with extensive experience in cloud infrastructure, automation, and system administration. Currently working at Times Internet Limited (Cricbuzz), I specialize in deployment automation, containerization, and monitoring solutions.


My expertise includes migrating large-scale applications from AliCloud to GCP, setting up comprehensive monitoring architectures, and automating deployment processes using tools like Ansible, Terraform, and Docker. I have hands-on experience with various cloud platforms and databases including Redis, MySQL, Cassandra, MongoDB, and Elasticsearch.


I hold multiple certifications including HashiCorp Terraform Associate, Microsoft Azure Fundamentals, and Red Hat Certified Specialist in Ansible Automation. I'm passionate about infrastructure automation, cloud technologies, and creating efficient deployment pipelines.
